SETUP INSTRUCTIONS:

+   Install latest version of python.
+   Install required modules using "pip install -r requirements.txt" command.
+   Rename the sample-config.json file to config.json.
+	Create a new telegram bot by following this --> (https://medium.com/shibinco/create-a-telegram-bot-using-botfather-and-get-the-api-token-900ba00e0f39)
		and get your bot token and put in config.json file.
+	Follow this --> (https://core.telegram.org/api/obtaining_api_id) to create telegram app 
		and get your api id and hash and put in config.json.
+	Enter your telegram username in config.json as admin username.
+   Save the config.json file.
+   Run the bot using "python main.py" command.
+   Let me know if you have some problem.
